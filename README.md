# vue-ast
练习vue的AST


### GoGoCode
当前 GoGoCode 支持解析和操作如下类型的代码：
1、JavaScript(JSX)
2、Typescript(TSX)
3、HTML
4、Vue

## 使用
GoGoCode 项目追求让代码转换一事变得简单，相较于同类，GoGoCode 提供了更符合直觉的 API：
1、一套类 JQuery 的 API 用来查找和处理 AST
2、一套和正则表达式接近的语法用来匹配和替换代码
